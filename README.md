# 🚗 Коды регионов РФ

[![Version](https://img.shields.io/badge/version-3.9.7-blue.svg)](https://github.com/drts/region-codes-rf)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![PWA](https://img.shields.io/badge/PWA-ready-purple.svg)](https://web.dev/progressive-web-apps/)

Современное PWA-приложение для определения региона России по номеру автомобиля. Стеклянный дизайн (Glassmorphism + Neumorphism), поиск по коду и названию, голосовой ввод, поддержка тёмной и светлой темы.

</p>

## ✨ Особенности

- 🔍 **Два режима поиска**: по коду региона и по названию
- 🎤 **Голосовой ввод**: большая стеклянная кнопка с эффектом пульсации
- 🎨 **Современный дизайн**: Glassmorphism + Neumorphism
- 🌓 **Три темы**: тёмная, светлая, автоматическая
- 📱 **Адаптивность**: корректно отображается на любых устройствах
- 💾 **Офлайн-работа**: данные кэшируются в localStorage
- 📋 **Детальная информация**: столица, телефонный код, все коды ГИБДД, ОКАТО, ISO, часовой пояс
- 🇷🇺 **Военные коды**: отдельное отображение для военных номеров
- 🔄 **Автодополнение**: подсказки при вводе

## 📦 Установка и запуск

### Вариант 1: Простой (браузер)

1. Скачайте все файлы репозитория
2. Откройте `index.html` в любом современном браузере

### Вариант 2: PWA (установка на устройство)

1. Откройте `index.html` в браузере на мобильном устройстве
2. Нажмите «Установить приложение» в браузере (Chrome/Edge/Safari)
3. Приложение появится на рабочем столе

### Вариант 3: WebToApp (сборка APK)

1. Скачайте [WebToApp](https://github.com/shiaho777/web-to-app)
2. Выберите тип «HTML / Frontend»
3. Укажите папку с файлами
4. Настройте package name, иконку и подпись
5. Соберите APK и установите на устройство

## 📊 Источники данных

- Актуальные коды регионов РФ (включая новые коды 100-199)
- Данные ОКАТО (Общероссийский классификатор)
- Коды военных ведомств (ФСБ, МЧС, ВДВ, военные округа и т.д.)

## 🛠 Технологии

- Чистый HTML + CSS + JavaScript (без библиотек)
- Web Speech API (голосовой ввод)
- localStorage (кэширование данных)
- CSS Custom Properties (темы)
- CSS Animations (плавные переходы)
